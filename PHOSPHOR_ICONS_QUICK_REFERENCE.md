## Icon Weights Available

Every icon listed above is available in 6 different weights:

1. **thin** - Ultra-light strokes (1.5px)
2. **light** - Lighter strokes (1.75px)
3. **regular** - Default weight (2px) [Most common]
4. **bold** - Heavier strokes (2.5px)
5. **fill** - Solid filled version
6. **duotone** - Two-tone style with transparency

## Usage Tips

1. **Consistency**: Use the same weight throughout your UI
2. **Size**: Common sizes are 16px, 20px, 24px, 32px, 48px
3. **Accessibility**: Always provide alt text or aria-labels
4. **Performance**: Import only icons you need
5. **Color**: Icons inherit text color by default (use CSS color property)

## File Naming Pattern

Icons follow kebab-case naming:
```
icon-name.svg              (single word)
icon-name-modifier.svg     (with modifier)
icon-name-style-size.svg   (with multiple modifiers)
```

Examples:
```
heart.svg
heart-straight.svg
arrow-left.svg
arrow-circle-left.svg
user-circle-plus.svg
```
